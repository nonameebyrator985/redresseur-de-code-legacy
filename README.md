# Redresseur de Code Legacy

Bienvenue dans le projet Redresseur de Code Legacy ! Cet outil est conçu pour aider les développeurs à analyser et refactoriser leur code hérité.

## Motivation
Le code hérité peut souvent être difficile à comprendre et à maintenir. Notre objectif est de simplifier ce processus et d'automatiser certaines améliorations structurelles qui peuvent rendre le code plus propre et plus performant.

## Fonctionnalités
- **Analyse de code** : Évalue la qualité du code en détectant les mauvaises pratiques et les points d'amélioration.
- **Refactoring automatique** : Propose des suggestions de refactoring avec des exemples de code.
- **Rapports** : Génère des rapports détaillés sur l'état du code et les améliorations effectuées.
- **Intégration continue** : S'intègre facilement dans les pipelines CI/CD pour des analyses régulières.

## Installation
Pour installer le projet, veuillez cloner le dépôt et exécuter les commandes suivantes :

```bash
git clone https://github.com/votre-utilisateur/redresseur-de-code-legacy.git
cd redresseur-de-code-legacy
npm install
```

## Utilisation
Pour utiliser l'outil, exécutez simplement la commande suivante dans votre terminal :

```bash
node index.js /chemin/vers/votre/code
```
